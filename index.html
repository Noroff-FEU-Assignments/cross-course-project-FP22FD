Web Security

How would you advise a potential client to secure his/her website from web security attacks? Note: the word count for your answer should not be greater than 250 words

There are many things you can do to have good digital security and some of the simplest are:

    1. choose a good hosting
    2. use secure passwords
    3. use a third party for secure authentication and authorization
    4. use firewall and malware detector and keep it up to date
    5. make backups regularly
    6. limit the amount of internet downloads
